# MathWiki: K-12 Digital Math Textbook

A premium, analog-style digital mathematics textbook project covering elementary, middle, and high school curricula.

## Project Structure

- `raw/`: Raw input sources (PDFs, notes, text).
- `vault/`: Organized knowledge graph (Obsidian Vault).
- `web/`: Astro-based publishing engine.

## Vision

To create a digital textbook that feels like a beautiful physical book—featuring EB Garamond typography, cream paper textures, and high-fidelity interactive math visualizations.

## Getting Started

1. Add raw materials to `/raw`.
2. Organize knowledge in `/vault`.
3. Preview the web site in `/web` using `npm run dev`.
