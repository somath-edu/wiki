# 🚀 AICE ASSOCIATE 실전 준비 로드맵

> [!abstract] National Public Certification
> **AICE ASSOCIATE**는 과학기술정보통신부가 공인한 **국가공인 민간자격**입니다. 단순 이론보다는 정형 데이터(Tabular Data)를 활용하여 실제 문제를 해결하는 **100% 실기 역량**을 검증합니다.

---

## 🏛️ 시험 개요 및 평가 기준 📊

합격을 위해서는 데이터 전처리부터 딥러닝 모델링까지의 전체 파이프라인을 90분 안에 막힘없이 구현해야 합니다.

| 항목 | 상세 내용 |
| :--- | :--- |
| **평가 방식** | 100% 온라인 실기 (Jupyter Notebook / Python) |
| **시험 시간** | 90분 (시간 안배가 핵심) |
| **문항 구성** | 총 14문항 (연속된 코딩 시나리오 수행) |
| **합격 커트라인** | **80점 이상** (100점 만점) |

## 💡 단계별 합격 전략 (Step-by-Step) 로드맵

### 1단계: 필수 라이브러리 익히기 🛠️
AICE 시험의 3대 도구를 먼저 마스터해야 합니다. 초보자라면 아래 상세 가이드를 반드시 읽어보세요.
*   **Pandas**: 데이터 요리 전 "재료 손질" (표 읽기, 데이터 확인)
*   **Scikit-learn**: AI에게 "공부시키기" (머신러닝 학습 및 예측)
*   **TensorFlow**: "인공 신경망 만들기" (딥러닝 레이어 구성)
*   👉 **[[AICE_ASSOCIATE_기초_라이브러리_가이드|📚 초보자를 위한 라이브러리 상세 가이드 보기]]**

### 2단계: 실전 전처리 파이프라인 구축 🧹
데이터를 AI가 학습하기 좋은 상태로 다듬고, 실제 시험 환경에 적응하는 단계입니다.
*   **결측치 처리**: "빵꾸난 데이터 메우기" (`isna()`, `fillna()`, `dropna()`)
*   **범주형 데이터**: "문자를 숫자표로 바꾸기" (원-핫 인코딩, `get_dummies()`)
*   **검정 대비**: "주피터 랩 생존법" (단축키 및 셀 실행 요령)
*   👉 **[[AICE_ASSOCIATE_전처리_및_환경_가이드|📚 초보자를 위한 전처리 및 주피터 랩 상세 가이드 보기]]**


### 3단계: 시간 관리 및 오픈북 전략 ⏱️
*   AICE는 **제한적 오픈북**입니다. 공식 라이브러리 문서를 빠르게 검색하는 법을 익히되, 코드를 처음부터 검색하면 시간이 부족합니다. 기본 구조는 암기해야 합니다.
*   배점이 높은 **머신러닝/딥러닝 모델링(40점)**에서 실수를 줄이는 것이 합격의 지름길입니다.
